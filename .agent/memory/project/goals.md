# Goals

## Mission
Build and maintain **DarkFact** — a provider-agnostic, self-improving agentic workspace template that enables autonomous project evolution across Claude Code, Gemini CLI, and OpenCode.

## Active Goals
- **Claude Code Harness**: Fix broken memory guards and implement PreCompact/Subagent hooks.
- **Self-Improvement Loop**: Deploy the Pain Point Monitor and automated version checking via `gh`.
- **Agent-Agnosticism**: Achieve full feature parity across all supported CLI providers.
- **Codebase Purification**: Remove legacy files and simplify the template structure.

## Success Criteria
- ✅ All 3 CLI providers achieve 100% feature parity.
- ✅ Downstream projects self-improve via the analyst/dev/maintainer loop.
- ✅ Zero-friction template updates via `make update-template`.
- ✅ Security guards successfully block out-of-scope edits.
